# Postman — MedCare API

Import file **`MedCare-API.postman_collection.json`** vào Postman.

## Cách dùng

1. Khởi động backend: `START-BACKEND.bat` hoặc `cd backend && npm start`
2. Trong Postman: **Import** → chọn file collection
3. Chạy **Auth → Login** (token tự lưu vào `{{token}}`)
4. Test các request còn lại

Base URL: `http://localhost:5000`

Tài khoản demo: `admin@demo.com` / `123456`
